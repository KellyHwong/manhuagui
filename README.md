# webp

webp 是 Google 的图像存储格式，Google Chrome 等浏览器可以识别和浏览。
漫画图片文件夹下自动生成了 index.html 模板，可以直接打开，里面实现了简单的图片浏览器。

# Usage

下载 别当欧尼酱了 第一话 https://www.manhuagui.com/comic/23552
manhuagui --id=23552 [--vol=1] # 默认只下载第一话
